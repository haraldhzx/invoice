# invoice service
Design a invoice scanning and classification engine with a server component and mobile app allowing to scan or manually enter invoices and categorize by type of expense. 
Multimodal LLM models can be used to analyze the invoice.

It shall also be possible to import supermarket bills and bank account transactions. 

The server side can run in the cloud. 

Add a llm module which allows to ask queries to the spend database via MCP such as
- how much did I spend on sweets?
- How much are house basic costs?#
- How much was spent for the kids? 

Implement the server side as spring boot application and for mobile app support both android and iPhone runtime. 

design as a markdown file for a set of epics and tasks with checkboxe.
